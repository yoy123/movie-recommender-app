# Privacy Policy Hosting

This directory contains the privacy policy HTML file for hosting on GitHub Pages.

## Setup GitHub Pages

1. Go to your repository on GitHub: https://github.com/yoy123/movie-recommender-app
2. Click **Settings** (top right)
3. Scroll down to **Pages** (in the left sidebar under "Code and automation")
4. Under **Source**, select **Deploy from a branch**
5. Under **Branch**, select **main** and **/docs** folder
6. Click **Save**

## Your Privacy Policy URL

After enabling GitHub Pages, your privacy policy will be available at:

**https://yoy123.github.io/movie-recommender-app/**

This is the URL you'll enter in Google Play Console when submitting your app.

## Verification

After setting up GitHub Pages:
1. Wait 1-2 minutes for deployment
2. Visit the URL above
3. You should see your formatted privacy policy

## Alternative: Custom Domain (Optional)

If you have a custom domain:
1. Add a CNAME file to this directory with your domain
2. Configure DNS settings with your domain provider
3. Update the custom domain in GitHub Pages settings

## Updating the Privacy Policy

To update the privacy policy:
1. Edit `docs/index.html`
2. Update the "Last Updated" date
3. Commit and push to GitHub
4. Changes will be live within 1-2 minutes
