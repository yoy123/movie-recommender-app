## Recommendations

### 1. Remove OMDB_API_KEY
**Reason:** Dead code (not used in UI)  
**Action:** Delete from `build.gradle.kts`, `local.properties`, `local.properties.example`

### 2. Externalize Hardcoded Constants
**Reason:** Trackers/mirrors may change, cache limit may need tuning  
**Action:** Move to `build.gradle.kts` or config file

### 3. Add Config Validation
**Reason:** Missing API keys cause silent failures  
**Action:** Add startup checks in `MovieRecommenderApplication.onCreate()`:
```kotlin
require(BuildConfig.TMDB_API_KEY.isNotBlank()) { "TMDB_API_KEY missing" }
require(BuildConfig.OPENAI_API_KEY.isNotBlank()) { "OPENAI_API_KEY missing" }
```
